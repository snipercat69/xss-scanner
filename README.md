# xss-scanner

XSS (Cross-Site Scripting) vulnerability scanner Discord bot. Crawls web pages, injects XSS payloads, and detects reflected/stored/persistent XSS vulnerabilities. Includes rotating proxy support for stealth scanning.

## Topics
python, discord, bot, xss, scanner, cybersecurity, vulnerability, pentest

## Setup

```bash
# Clone the repo
git clone https://github.com/snipercat69/xss-scanner.git
cd xss-scanner

# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
python3 xss-scanner.py
```

Built with [OpenClaw](https://github.com/openclaw/openclaw) — an AI agent framework. This app runs as a Discord bot / scheduled automation powered by OpenClaw's cron and channel integration system.
